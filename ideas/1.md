# 邮件发送模块设计方案

## 一、用户需求

在聊天模块中实现邮件发送功能，通过技能插件方式注入。

### 交互流程

```
用户: /发邮件
    ↓
Latte Agent: 显示邮件发送表单（技能模板）
    ├── 邮箱地址: [下拉选择 / 新增]
    ├── 内容: [AI 文本编辑器]
    └── [执行命令] → 弹框确认 → 本地执行
```

---

## 二、技术实现

### 2.1 技能插件注册

**文件结构:**
```
~/.openclaw/skills/send-email/
├── SKILL.md          # 技能描述
├── template.json     # 表单模板定义
└── scripts/
    └── send.sh       # 发送脚本
```

**SKILL.md 触发词:**
- `/发邮件`
- `/发送邮件`
- `/send mail`
- `/email`

### 2.2 表单模板 (template.json)

## 四、前端组件实现

### 4.1 组件结构

```
src/
├── components/
│   ├── SkillForm.vue       # 通用技能表单组件
│   ├── EmailForm.vue       # 邮件发送表单（基于 SkillForm）
│   ├── AiTextEditor.vue    # AI 文本编辑器弹窗
│   └── CommandConfirm.vue  # 命令确认弹窗（已有）
├── views/
│   └── ChatView.vue        # 聊天视图（集成技能触发）
└── utils/
    ├── skill-parser.ts     # 技能模板解析器
    └── command-builder.ts  # 命令构建器
```

### 4.2 核心接口

```typescript
// 技能模板接口
interface SkillTemplate {
  id: string
  title: string
  fields: FormField[]
  command: string  // 模板字符串
}

// 表单字段类型
type FieldType = 'text' | 'textarea' | 'select' | 'select-or-input' | 'ai-editor' | 'file-picker'

interface FormField {
  id: string
  label: string
  type: FieldType
  placeholder?: string
  options?: string[]
  default?: string
  required?: boolean
  aiAssist?: boolean  // AI 辅助开关
}

// 表单数据
interface FormData {
  [fieldId: string]: any
}
```

### 4.3 技能触发流程

```typescript
// ChatView.vue 中处理技能触发
const handleSkillTrigger = (skillId: string) => {
  // 1. 加载技能模板
  const template = await loadSkillTemplate(skillId)
  
  // 2. 显示表单弹窗
  showSkillForm(template, (formData) => {
    // 3. 构建命令
    const command = buildCommand(template.command, formData)
    
    // 4. 显示确认弹窗
    showCommandConfirm(command, () => {
      // 5. 执行命令
      executeCommand(command)
    })
  })
}
```

## 五、扩展思考

### 5.1 技能通用化

邮件发送只是一个例子，这套机制可以扩展到其他技能：

| 技能 | 触发词 | 表单字段 |
|------|--------|----------|
| 发邮件 | /发邮件 | 收件人、主题、内容、附件 |
| 发消息 | /发消息 | 平台、对象、内容 |
| 创建任务 | /新建任务 | 标题、描述、截止日期、优先级 |
| 生成报告 | /生成报告 | 类型、时间范围、格式 |
| 部署项目 | /部署 | 环境、分支、配置项 |

### 5.2 AI 智能化增强

**内容字段 AI 增强:**
1. **润色** - 优化文字表达
2. **翻译** - 中英互译
3. **总结** - 提炼要点
4. **扩写** - 丰富细节
5. **语气调整** - 正式/轻松/紧急

**智能推荐:**
- 根据聊天上下文自动提取邮件主题
- 根据历史记录推荐收件人
- 根据时间自动填充日期变量

### 5.3 安全机制

1. **敏感操作确认** - 所有命令执行前弹框确认
2. **命令白名单** - 只允许执行预定义的命令模板
3. **输入校验** - 邮箱格式、文件路径等验证
4. **审计日志** - 记录所有命令执行历史

### 5.4 多渠道发送

除了邮件，可以扩展到：
- 微信消息
- 钉钉通知
- Slack 消息
- Telegram 消息
- 短信
